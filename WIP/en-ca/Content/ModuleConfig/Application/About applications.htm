About applications

Clients submit applications using an online portal. Once they have completed all required questions and fields and click Submit:

  • Data fields and responses to questions on the application are transferred to a GCPM application.
  • A snapshot of the data is captured in a non-editable PDF that is stored in a location that is accessible to the client as well as staff at your organization.
  • If priority scoring has been configured for applications, a priority assessment is automatically attached to the application.

The application status is a configurable lookup, and can be built into business processes to trigger communication or other activities. The internal status of an application can be mapped to an external status for clients.

For example, a client could see "application received" and the internal status could be one of these: "received / under review / converted to project"

Applications are configured both in GCPM (using Dynamics CRM) and in the portal (using Alphinat software). When a client chooses a funding opportunity to apply to, the portal displays an application, populated with data fields from the appropriate GCPM application template. Read more about the portal here.

An applicant's responses are assessed for eligibility and value. Responses can be scored automatically to facilitate workload management, particularly for high volume programs. See About scoring (ranking) applications.

The WES program in GCPM provides an example of a priority assessment on applications.

Applications submitted by email, fax or other manual options can be manually entered.

Localizing the application module
  • Your organization must determine what information is on an application form, and how to manage this in CRM.
  • Your organization must decide how to manage incomplete applications, duplicate submissions, and applications submitted in error or withdrawn. Manual administrative action is required to associate an application to a project, and to a GC Client (or create new records, if applicant is new).
  • Your organization can use work flow for validated clients (those with an existing account and GC Client record) to have their applications registered as a project.
