Best practices: assessment questions

It is a best practice to map out your assessment process before finalizing assessment questions, to take advantage of the automation and efficiencies that assessment questions can provide.

Here are some ideas to use assessment questions effectively in your organization:

Display questions in appropriate locations

Do you want to display questions in a single area, or sort them into grids, according to section title?

For example, questions about cost items can be displayed below the cost item grid.

  • Use the Section title field to sort into grids.
  • Be sure grids are configured in each section of the form where the assessment questions will be displayed.
  • Text responses and lookup values cannot be displayed in the same Crowe grids. They must be in separate grids.
Control which questions appear on projects

For projects that meet specific criteria, display additional questions. Form sections can be configured to dynamically show (or hide) based on responses.

For example, "Is your organization for-profit or non-profit?"

  • = for-profit, display additional questions related to for-profit entities in a separate Crowe grid.

  • = non-profit, hide for-profit questions.
  • Alternatively, grids can be configured to appear in different forms (e.g. one form for for-profit, one form for non-profit).
Display data in GCPM based on assessment results
  • Use the Question ID field to dynamically show / hide form sections based on the response to an assessment question.
  • For example, if the question "Project costs are over $5M" = "No" then hide the section on the form that has questions for projects over $5M.
Reference assessment results in documents

Use assessment questions to ensure officers' assessment is reflected in communications with the client, the contribution agreement and into the monitoring phase of the project.

  • Note: the Question ID field is alphanumeric (can be made up of letters and numbers) but should not contain a space.
  • Reference the Question ID field in XML document templates.
  • For example, the question "Does the project require a change of control?" is assigned Question ID = "COC".
    In the contribution agreement XML template, the clause associated with change of control is referenced with "COC". Insert appropriate code to display the correct clause with the associated "COC" response. The document generated from the template will display the appropriate clause based on the response to this question.
When to create multiple assessment templates

One assessment template can address multiple assessment factors, rather than creating multiple assessment templates (e.g. for-profit or non-profit client).

However, if there are more than one or two conditional assessment factors, this can lead to complex configuration for a single template. If there are more, it is advised to make separate templates. For example: 

  • Project costs are over / under $5M

  • For-profit and non-profit entities
