Configure an assessment question template

Assessment templates ensure the same assessment questions are copied to every project in a program.

To configure an assessment template:

  1. In Advanced Find, search for Assessment Question Template | A and create a new record.

  2. Name: Type the name of the template.
  3. Assessment questions: Click the + icon to add questions to this template.

  4. Add questions to the template.
    Note: assessment template elements have these fields: 
    • Question type: Indicate if the question is required or discretionary. Permits display in separate grids (discretionary / mandatory).
    • Mandatory: Indicate if a response is required for this question.
    • Answer type: Indicate if the response to the question is text, or a selected value.
    • Section title: Display the question below related topic (e.g. project costs, financing)
  5. Save and close the record.
