Key steps to configuring performance

Performance measurement is the key component to record success of a project.

To configure performance measurement:

  1. Navigate to Business Administration > Programs - Common > Priority Program | A and click + NEW to create a new Priority Program record.
  2. Fill in the fields using onscreen help and the notes below: 
    • Name (EN) and Name (FR): Enter a name for the priority program.
    • Enter a Name (EN) and Name (FR) for the Outcome.
  3. In the Program Outcomes grid, select + NEW to create a new outcome.
  4. Enter a name and save the program outcome record.
    Once the record is saved, you can add project indicators in the grid.
  5. Click + NEW to add a project indicator. Alternatively, select an existing indicator. Provide a name, along with descriptions and means of verification.
  6. When all indicators are associated to a program outcome, and all program outcomes are associated to a priority program, you have completed the configuration of performance measurement.

Project indicators are displayed in the Project Result section of the project form. The initial information established during project evaluation of the project as well as updates during the administration and monitoring stages of the project. Historic records are generated each time a change or update is made to a project result.
