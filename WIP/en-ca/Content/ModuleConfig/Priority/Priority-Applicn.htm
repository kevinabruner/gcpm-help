Configure automated priority ranking for applications

Note: this article assumes you are familiar with configuring Priority Assessment Templates. See this set of articles for more details: About priority assessment.

Automated priority ranking allows for client responses in an application to be automatically scored in a priority assessment. Each response option (stored in the Answer Options entity) to application question is assigned a score. The total score for an application is calculated automatically for an automated priority assessment. Only questions that have answer options configured can be used for an automated priority ranking.

The configuration is done in two steps: 

  1. Priority template element: For every priority template element that you would like auto-populated, select the application form question that will auto-populate this priority element.
    1. In the Priorities grid, review the priority template elements.
    2. For each priority that should auto-populate from an application, Edit the priority template element.
      See Create priority template elements for more detailed instructions.
  2. Scoring value: 
    1. Verify that the field Application Question is the appropriate question from the application. This ensures the field will be auto-populated with the input from the client's application. Verify that a scoring value is selected for each Option.

      See Application options for more detailed instructions.
    2. The scoring option should correspond to both the application question option and the priority template element.

  3. Create a priority assessment. See Assess project priority .

    The assessment generates several priority assessment elements, if there are any application questions that have an answer that corresponds to this element's answer. If so, then auto-pick that value. The automated scoring will calculate the priority assessment score. If all questions are configured to be automatically populated by the client's application, you have a fully automated priority ranking solution.

To view the rankings of all applications in a funding opportunity: 

  1. Navigate to the funding opportunity associated with those applications.
  2. In the top navigation bar, navigate to Applications.
