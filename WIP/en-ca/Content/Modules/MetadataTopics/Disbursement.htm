Disbursement module metadata

The first table below identifies entities in the disbursement module. The remaining tables display the egcs metadata for each entity. This metadata was generated from the Workspace 9 environment on Jun 18, 2019.

Disbursement module entity list

Entity Description Logical Name
Advance Disbursement feature - Request for an advance of funds submitted by a client egcs_fc_advance
Advance Item Disbursement feature - A line item that describes a portion of the request for an advance. Usually one line for each applicable budget category. egcs_fc_advanceitem
Claim Disbursement feature - Claims that have been submitted by a GC Client egcs_fc_claim
