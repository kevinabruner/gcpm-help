Document manager metadata

The first table below identifies entities in the document manager tool. The remaining tables display the egcs metadata for each entity. This metadata was generated from Tools 9 environment on Jun 18, 2019.

Document manager tool entity list

Entity Description Logical Name
CFG Document Entity A configuration entity that contains document management configuration for a specific entity and document repository. egcs_cfg_documententity
CFG Document Location A configuration entity that represents a folder or a predefined location in the external document repository. egcs_cfg_documentlocation
CFG Document Type A configuration entity that identifies a document as a specific type of document. egcs_cfg_documenttype
CFG Document Field A configuration entity that represents a single metadata field in the document repository that maps to a field in CRM. egcs_cfg_metadatafield
GCPM Document A document in the document repository. egcs_document
GCPM Document Location Document manager feature - An entity that identifies a folder for the storing of a document. egcs_documentlocation
GCPM Document Version This entity has not been implemented egcs_documentversion
