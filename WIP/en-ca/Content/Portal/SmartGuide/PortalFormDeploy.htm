Deploying forms for the portal with SmartGuide

There are several components involved in deployment of SmartGuide forms. They include:

  • Smartlets (the forms)
  • Connections (includes web services and extension functions)
  • Themes
  • Supporting dlls

When exporting and importing forms from server to server, all components (except supporting dlls) can be automatically included with Smartlets.

Note: When updating forms, by manually changing them or by importing from a different environment, it is important to compile the server-side Smartlet by viewing the Smartlet through SmartGuide Designer.

API – extension functions

When developing extension functions through SmartGuide’s API, be sure to take these steps:

  • Add new functions to apn-extension-function.xml, along with any input and return parameters.
    See SmartGuide_7.1_DeveloperGuide, page 6, for more details on developing and deploying extension functions.
  • After updating the DemoUtils.cs file and the above xml file, build the project, navigate to EGCS.Portal\EGCS.Portal.SmartGuide\bin\Debug and upload the DemoUtils.dll into the SmartGuide Designer. See SmartGuide_Designer_7.1_UserGuide, page 115, for more detailed instructions.
  • Additional assemblies used in any API extension functions need to be placed in C:\inetpub\wwwroot\smartguide\7.1\smartlets\bin and C:\inetpub\wwwroot\smartguide\7.1\sgd\bin.

Deploying forms from server to server

Importing SmartGuide content from one server to another is best done using the SmartGuide designer.

Exporting Smartlets

Export Smartlets either individually or by group:

  1. Log in to SmartGuide Designer on the source server and navigate to the Smartlets tab.
  2. Click the Export button to export Smartlets in current folder and subfolder(s) OR export a Smartlet or folder individually by navigating and using the export icon on its row entry.
    Exporting copies supporting files and Smartlets into a compressed folder.
Importing Smartlets

Similar to exporting, importing can be done individually or by group.

  1. Log in to SmartGuide Designer on the destination server, and navigate to the Smartlets tab, and click Import.
  2. Choose yes to import any changes to theme or connection files from the compressed folder that was downloaded in the export step. Note: this overwrites resources.

See SmartGuide_Designer_7.1_UserGuide pages 147-148 for additional info on exporting and importing Smartlets.
