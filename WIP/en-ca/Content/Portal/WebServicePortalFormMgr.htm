FormManager web service overview

The FormManager web service class contains multiple web services that either communicate with the portal or to CRM through RDA web service calls from SmartGuide forms.

Instructions on adding web service references to SmartGuide can be found at Developing forms for the portal with SmartGuide.

FormManager web services used in SmartGuide forms
Web services used in SmartGuide forms

Web service

Parameters

Returns

Description

LoadDraft

string ReferenceNumber

string XML

On Smartlet Initialization, queries the current form instance’s SubmissionSoumission document set for XML to deserialize and populate into Smartlet fields.

SaveDraft

string ReferenceNumber,

string FormXml,

string FormPdfBase64,

bool IsSubmit

bool result

On Save button click or master page language toggle sends the serialized FormXml and referenceNumber to the current form instance’s SubmissionSoumission document set. FormPdfBase64 is blank and IsSubmit is set to false.

Returned results triggers the SaveResult field to inform the user of success or failure.

On Submit button click the generated FormPdfBase64 encoded string is included and IsSubmit is toggled from false to true.

If returned result is true advances the user to Successful Submission page that displays their ReferenceNumber.

UserCanEditForm

 

bool result

On Smartlet Initialization, returns whether or not the current user has the appropriate permission to save and submit the form.

GetProjectOutcomes

string FC_Profile_ID

Data.FCProfile.egcs_fc_result[]

On Performance Measurement Report Smartlet Initialization, the Smartlet calls the portal GetProjectOutcomes web service which queries the RDA web service GetProjectOutcomes and returns an array of data objects that reflect the Project Outcomes that are available to funding case profile whose ID was provided.

GetProgramsList

 

Data.Program.egcs_TP_Profile[]

Is currently not used in any forms, however provides a list of currently available programs in case there is a desire to have a Smartlet pull the data from CRM versus having it hard coded into a Smartlet dropdown.

GetFCProfileID

 

FCProfileID

On Project Level (Claims, Advance, and Performance Measurement Report forms) Smartlet Initialization retrieves the Funding Case Profile ID.

Note: is returned as an FCProfileID and not a string as SmartGuide has its SharePointConnections configured as JSON return values.

GetBudgetCategories

string FC_Profile_ID

Data.Claim.egcs_FC_BudgetItem[]

On Smartlet Initialization on Claim and Advance Forms, queries the RDA Web Service GetBudgetCategories which returns an array of data objects used to populate the available Budget Category dropdown options in the claim items and advance items tables in the claim and advance forms, respectively.

GetProgramQuestionsFromCRM

Guid applicationFormIdGUID

Data.FormManager.egcs_ApplicationForm

On dynamic application form, when toggling from page 1 to page 2, the form calls GetProgramQuestionsFromCRM which passes the supplied GUID applicationFormIdGUID (which is set by the Program Drop down on page 1 to the RDA web service GetProgramSpecificQuestions if the current form program has been changed, thus requiring an update to the dynamic questions from CRM being displayed on page 2.

The returned object contains an array of questions and their appropriate metadata to construct and append SmartletFields along with the GUID representing the template returned to the form.

Web services used in SmartGuide forms

Web service

Parameters

Returns

Description
CreateDraft int FormDefinitionId string result

Called from the Master Page Create dropdown, creates an instance of a form whose type is determined by the provided FormDefinitionId in the current Portal Scope.

The returned result is the URL that the page is redirected to which Renders the newly created draft in SmartletRender.aspx
RemoveDraft string ReferenceNumber SubmissionRemoveResult

Called from the Drafts Web Part on the Recent Activity Page, removes the draft corresponding to the provided referencenumber.
ListDraft  

SubmissionListDraftResult

Called from the Drafts web part on the Recent Activity Page, lists all of the available draft forms to the current web. Information provided includes reference number, type, created data, created by, modified data, modified by – along with links to edit the draft and remove the draft.
