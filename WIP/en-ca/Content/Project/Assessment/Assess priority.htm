Assess project priority

This topic describes how to manually complete a priority assessment of an application or project.

To configure automated application scoring based on data in the application, see Configure automated priority ranking for applications.

To assess project priority:

  1. In a project, navigate to the Application Received chevron and open the Prioritization section of the form.
  2. If a priority assessment has already been completed, it is displayed in Latest priority assessment field.
  3. Click + in the grid. A search bar appears - click +New to create a new priority assessment record.
  4. A new priority assessment record displays, with the project name already associated. Fill in the fields using onscreen help and the notes below.
    • Project: Displays the project associated with this assessment.
    • Priority template: Choose the priority template for this assessment. The template provides the priority criteria and weighting.
    • Rollup formula: Displays the rollup formula used for this assessment.
    • Priority rating: Displays the score that is automatically calculated, based on the criteria responses and the rollup formula.
    • Adjusted rating: Optional - an officer can manually override the automatically assessed score.
    • Comment: If the score is adjusted, a comment is required.
    • Priorities grid: Displays the priorities for the assessment. When the criteria have been assessed, scores for each priority will display in the grid.
    • Criteria grid: Displays one or more criteria for each priority. Choose the appropriate response for each criteria. Depending on how the priority template is configured, scores for each response will provide the overall priority rating.
  5. Review the priority rating, and adjust the results if required. Include comments for any adjusted scores.
  6. Save and close the record.
