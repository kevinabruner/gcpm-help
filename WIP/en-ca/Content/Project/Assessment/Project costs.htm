Enter project cost items

Project costs are cost items identified during project assessment. The costs include all expenditures necessary to complete the project: eligible, eligible not funded, and ineligible costs. The department amount (the amount the RDA will support) is the calculation of eligible costs X rate of assistance.

A completed cost item table calculates the recommended amount of departmental funding for approval, as well as the total eligible, eligible not supported, and ineligible costs. A project cost grid is converted into a budget grid upon approval of the project.

To enter project costs for a project: 

  1. Navigate to the Project Costs section in the project assessment form.
    The project costs table is displayed.

    You can navigate to Cost items in the site map, but note, the grid here does not calculate totals, and requires data entry in the form, there is no grid option.

  2. Above the grid is the field: 

    • Effective rate of assistance: The calculated average rate of assistance for the overall project.
      It is a calculated field (total departmental amount / total eligible costs).

  3. Use the grid or form to add new cost items or modify existing ones. Use onscreen help and the notes below: 

    •  Cost category: Choose an appropriate cost category. Note that costs in the budget will be summarized at the cost category level, unless budget element is selected.
    • Cost subcategory: Choose a subcategory, which is related to the category selected above.
    • Fiscal year: An optional field that can be used if certain costs are to be incurred within a specific period for a multi-year project.
      (Note: displays in the form, but not the Crowe grid.)
    • Item detail: Provide a short description of the item, e.g. name of the equipment
    • Total costs: Enter the total cost of the cost item.
    • Eligible costs: Enter the amount of the total that is eligible for funding
    • Eligible not funded: Enter the amount of the total that is eligible for funding but not funded
    • Ineligible costs: Enter the amount of the total that is ineligible for funding
    • Rate of assistance: auto-calculated (Departmental Amount / eligible costs (or 0 if eligible costs =0).
    • Departmental amount: Type in the departmental amount (the recommended amount the agency will pay).
      Note this amount must be less than or equal to eligible costs.
    • Budget element:  Items indicated as budget elements appear as a separate line item in the project budget, and are not included under a category. If you do not check this field, the item will be included in the cost category / subcategory total.
    • Locked: a cost line is locked when the project has been approved. Changes to cost items after project approval must be completed using the amendment process.

    Note: Cost items must have a cost category and a total cost before you can save them. The other fields are optional.

  4. Continue adding all project costs, then save changes to the project.

The totals for the cost items are displayed in the "Client Profile and Project Summary" tab.

  • Total costs: Must be equal to (Eligible Costs + Eligible not funded + Ineligible costs ) in the project costs grid.
  • Total eligible costs: Formula = sum of all eligible costs in the project costs grid.
  • Total departmental amount: Formula = sum of all departmental amounts in the project costs grid.
