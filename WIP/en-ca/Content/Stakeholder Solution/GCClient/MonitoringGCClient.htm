GC Client monitoring

Client monitoring can occur as long as the client has an active project(s). In performing monitoring activities, officers exercise professional judgement, especially when dealing with unique situations that occasionally occur.

To add a monitoring activity to a GC Client: 

  1. In a GC Client record, navigate to GC Client Monitoring.
  2. Add a new client monitoring.
  3. The client monitoring form displays. Fill in the fields using onscreen help and the notes below.
    • GC Client Monitoring Template: Choose the appropriate template for the client. The template generates monitoring requirements and checklists for the client.
    • Monitoring status: Indicates the status (in progress / completed)
    • Monitoring type: Choose the frequency (annual / semi-annual / quarterly / monthly / ad hoc).
    • Client risk review: Select an existing review for the client or create a new one by clicking + New.
      SeeCreate a client risk assessment for further instructions.
    • Financial review: Select an existing review for the client or create a new one by clicking + New.
      See Review two or more financial statements for further instructions.
    • Any risks / obstacles: Provide any comments or supporting information required.
    • Notes: Provide a note, if applicable.
    • Next review date: Enter the date of the next monitoring activity.
    • Next financial date: Enter the date for the requirement of the next financial statements.
    • Date completed: The date is auto-populated when the monitoring status = completed.
  4. Checklist: Answer questions in the checklist. The questions are program-specific.

  5. When the monitoring is completed, change the status to Completed.

  6. Save and close the record.
  7. Save and close the record.
