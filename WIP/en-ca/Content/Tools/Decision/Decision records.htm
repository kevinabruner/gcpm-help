Decision records

To implement the decision framework for an entity, there must be a separate entity to save the decisions. The name convention for this entity is [ENTITY] Decision.

A decision record has the following fields to capture key data for a decision:

  • Entity: References the entity record this decision belong to. E.g. a disbursement decision record has a reference to a disbursement record, while a project decision record has a reference to a project or an amendment record.

  • Decision status: Displays the most recent status of a decision. For example, if a user starts an approval process, the status is in progress. If one user denies, the status is denied. If the last approver approves, the status is approved.
  • Decision date: date the decision was made.
  • On behalf of: if the decision was delegated, identifies the person meant to make the decision).
  • Rationale: if any comment or reasoning is provided by the decision maker, it is tracked here.
  • Approval process: Contains approval action records related to the approval process, based on the CFG Authority Type and CFG Authority Set selected in the transition path.
  • Related documents: any documents that were automatically created as part of the decision (the ones set in the related Decision Type record), or manually added by the decision maker, appear here.

Project Decision records are the only ones that can reference two entities.

Entities that have decisions
Entity [Entity] Decision

Description
Application Application Decision Presently only used for the automatic logging of a decision whenever an application's status changes.
GC Client Client Decision Logs decisions related to a GC Client such as On Hold, Recovery, Ready for Funding.
Disbursement Review Disbursement Decision Logs decisions related to a Disbursement Review such as Section 34.
PAYE PAYE Decision Logs the approval of a PAYE.

Project and Amendment

Project Decision

Logs all decisions related to a project including any amendments to that project.

  • GC Client
  • Disbursement Review
  • PAYE
  • Project and Amendment (both use the Project Decision entity).
