Create web resources

Three web resources must be created. The web resources are for three different entities: 

  • [Entity] Decision form,
  • [Entity] History form
  • [Entity] ribbon.

For information on creating TypeScript script and its JavaScript counterpart, see Framework (for TypeScript).

Create a web resource to be used in the [Entity] ribbon: 

  1. Create a web resource and name it egcs_/entities/[ENTITY LogicalName]/ribbons/form.js.
    For example: egcs_/entities/egcs_fc_disbreview/ribbons/form.js.
    Source code location:$/GCPM/EGCS/EGCS.CRM/EGCS.CRM.Framework/EGCS.CRM.WebResources/egcs_/entities/egcs_fc_disbreview/ribbons/form.js
  2. Copy the code from another entity’s form.tsthat has transition. (e.g. use the Disbursement Review entity).

Create a web resource to be used in the [Entity] Decision form

  1. Create a web resource for the [Entity] Decision and name it egcs_/entities/egcs_[ENTITY]_decision/forms/default.js

    For example: egcs_/entities/egcs_disb_decision/forms/default.js.
    Source code location: $/GCPM/EGCS/EGCS.CRM/EGCS.CRM.Framework/EGCS.CRM.WebResources/egcs_/entities/egcs_disb_decision/forms/default.ts

  2. Copy the code from another decision entity’s default.ts, for example disbursement decision.
  3. Modify the following lines to use the values of your new entity.
    Mandatory changes are in red. You can also change the name of the variable disbursement, but it is not necessary for it to work.
  4. Add egcs_/shared/js/main.js script to [Entity] Decision form libraries.

  5. Add egcs_/shared/js/main.js script to Onload form event and pass the name of the script created in this step as parameter.
    Note: this parameter should not contain the .js suffix. Check the option to pass execution context as first parameter.

Create a web resource to be used in the [Entity] History form

  1. Create a web resource to be used in the [Entity] History form called egcs_/entities/egcs_[ENTITY]_history/forms/default.js

    For example: egcs_/entities/egcs_disb_transitionhistory/forms/default.js.
    Source control location: $/GCPM/EGCS/EGCS.CRM/EGCS.CRM.Framework/EGCS.CRM.WebResources/egcs_/entities/egcs_disb_transitionhistory/forms/default.ts

  2. Copy the code from another [Entity] History: default.ts, for example, disbursement history.
  3. Modify these lines to use the values of your new entity.
    Mandatory changes are in red. You can also change the name of the variable but it's not required for it to work.

  4. Add egcs_/shared/js/main.js script to [Entity] Decision form libraries.

  5. Add egcs_/shared/js/main.js script to Onload form event and pass the name of the script created in this step as parameter.
    Note: this parameter should not contain the .js suffix.

  6. Check the option to pass execution context as first parameter.
