About the document generator

The document generator can generate any document your organization may need to manage clients and projects. Business users can modify document templates to display CRM data. The document generator is a tool that takes snapshots of data and converts it to human-readable format (e.g. a document, a report or web page). The document generator provides flexible formatting and content design, and can readily tailor documents to reflect the look and feel of your organization.

Document templates can be configured to display project-specific information. Conditional text, such as clauses determine by assessment outcomes and project data, is configured using XML code that is linked to GCPM fields.

Example of a conditional clause: There are three clauses related to environmental regulations, but only one clause can be chosen for any given project, based on characteristics of the project. When an officer identifies the suitable option for a project, the appropriate clause is inserted into the CA for that project.

A configurator defines the data fields to include in a snapshot, and the snapshot populates an MS Word document template with data in XML format.

GCPM includes a configured CA template.

Features: 

  • Create snapshots: (in XML or JSON formats)
    • A point-in-time picture of data
    • The basis to generate documents (or reports) 
  • Generate documents (in Word, HTML, PDF formats)
Technologies required:
  • Microsoft Word XML mapping (available in MS Word 2013 and above)

Useful tools (others can be employed):

  • Open XML SDK: open source library from Microsoft that allows you to programmatically create and edit Office documents without installing Microsoft Office.
  • Open XML Power Tools : Open source library from MS that adds extra functionality to the Open XML SDK, such as building and assembling documents and convert Word to HTML
  • XPath 2.0: Used by OpenXML Power Tools to insert advanced dynamic content in MS Word.
