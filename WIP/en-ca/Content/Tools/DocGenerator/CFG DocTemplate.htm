Configure document templates

Configure document templates to make Microsoft Word templates available to the GCPM Document Generator.

To configure a document template: 

  1. In advanced find, search for CFG Document Template
  2. Use the onscreen help to configure the record. See field descriptions below for further instructions.
    • Name: The name of this document template. This is the name that will appear to end users when they choose a document to generate.
    • Language: Specify the language of the document template. Choose en-ca or fr-ca.
      • This indicates the language the template will use to populate the document, where fields have been configured bilingual.
      • Language choice can also impact formatting of certain values in the document, such as currency values, e.g. $2,000.00 (EN) vs 2.000,00 $ (FR).
    • Formatting: To override the settings in the snapshot, modify the settings below: 
      • Decimal places: The number of decimal places for decimal numbers only.
      • Date format: The format of the date in datetime fields. CRM defaults to yyyy/MM/dd.
      • Time format: The format of time in datetime fields. CRM defaults to hh:mm:ss.
      • Currency format: The format of currency fields.
  3. Templates: Click inside the Notes field, and click Attach to upload a template.

    • Provide a brief note or title to indicate the purpose of the template.
    • Document templates are versioned. The latest version is used to generate documents.
  4. Click Done to upload the template.
  5. Save and close the record.

Once document templates are configured, you can attach them to program profiles.
