Using document templates

To view the document generator templates available in a project: 

  1. Open the a record and click the Document Generator button in the top menu bar. This button may be in the ellipses menu (...).

  2. View or download the document.
    • View: provides an HTML rendering of the document, including formatting.
    • Download: inserts the data into the template, and downloads a Microsoft Word document locally.

When to configure OOB document templates vs. using the document generator?

OOB Document Templates are for simple scenarios:

  • To generate a document with few fields, that are directly related to the entity for which you are building the template.

Document generator allows for greater range of data and flexibility in the displayed content.

  • Can be generated manually or automatically.

  • No limit on the number of levels away for related data.

  • Can contain conditional content.

  • Can use XPath.

  • Reusable in multiple entities.

  • Templates are reusable.

  • Data sources are reusable.

  • Not associated only to an entity.
