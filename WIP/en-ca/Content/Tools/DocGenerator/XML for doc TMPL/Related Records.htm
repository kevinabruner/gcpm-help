Related records

Note: As of May 27, 2019, this method is currently broken. It is best to use the Repeat or Table tag.

Referencing and inserting a set of related records can be accomplished in a few ways. You can insert it as a list of items, as a table of items, or any other type repeating content.

However, this method is best used for generating a table of items because the other types are not as straightforward. For producing something other than tables, refer to the Repeat section inFormatting an XML template document .

E.g., referencing / inserting a table of project cost items.

  1. Insert an empty table with two rows and however many number of columns. The first row is the header row and the second row is the data row that will increase based on the number of rows of data.
  2. Highlight the entire second row of the table (it is blank):
  3. Go to the XML Mapping pane and find the list of records you wish to tabularize and insert it as a “Repeating” content control.
  4. In the XML Mapping pane, insert field data that belong to the list of records into each table cell.

Displaying Table Totals

  • Word can sum table columns, but Word security makes it so that the summing does not occur until you open the document and click OK.
  • This won't work if you are using the table in a document that is immediately uploaded to a records repository and possibly never opened.
  • If you are creating a table and require a totals row you will need to use CRM to perform the rollup / calculation first.
  • For example, if you want to show a table of project costs in your contribution agreement and have the last row display the column totals you will need to go to CRM and add new fields on the project entity for each value that you want to display. Then set up rollup configurations to populate those fields (or calculation configurations, whichever suits your scenario).

There may be other ways to accomplish this, but it will most likely involve your development team. Please discuss with them if the above solution is not acceptable.
