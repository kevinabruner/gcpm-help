Solution settings for the document manager

To configure the Document Manager settings: 

  1. Open the imported GCPM_CFG_DOCUMENT_MANAGER solution by navigating toSettings > Solutions > GCPM_CFG_DOCUMENT_MANAGER. Solution settings are found in Configuration (left hand menu).
  2. Fill in the appropriate settings the document management system used at your organization.
    Settings for GCDocs and Sharepoint are listed below.
  3. Click Save Changes.

Notes: 

  • If you create custom plugin steps and want the solution settings saved automatically to theSecure Configuration field, set the Unsecure Configuration field of the plugin steps to the following GUID: bf8b5840-4578-491a-9c92-e37016e1ecea
SharePoint settings

The settings in the table below are used by the SharePoint connector plugin.

Note: If you override the authentication mechanism, you may choose to store your settings elsewhere. For example, you could disable the authentication plugin step and register the authentication plugin step under your own custom plugin instead.

SharePoint settings

Setting

Description

Server Url

The domain of the SharePoint site, which will usually point to the Root Site Collection.

For host-named site collections, the “domain” will also be the site collection url.

Site Url

The url of the site collection. You are not limited to just one site collection, but at least one must be provided. This url may be the same as the Server Url.

Relying Party Trust

The Relying Party Trust identity that CRM uses that is trusted by your SharePoint ADFS server.

Username

The user name of the admin service account.

Password

The password of the admin service account.

Max Folder Length

The global max length of a folder name, which can be overridden. Names will be truncated.

Max File Length

The global max length of a file name, which is can be overridden. Names will be truncated.

GCDocs Connector settings

Setting

Description

Site Url

The GCDocs site url.

CWS Url

The GCDocs content web service endpoint.

Root Folder Id

The node id of the root folder. This is the starting point for all folders that will be created. Optional.

Username

The user name of the admin service account.

Password

The password of the admin service account.

Max Folder Length

The global max length of a folder name, which can be overridden. Names will be truncated.

Max File Length

The global max length of a file name, which is can be overridden. Names will be truncated.
