About dynamic tabs

The dynamic tab plugin is designed to hide (or show) data fields on a form based on the context of the data. Dynamic tab records are configured to display data relevant to a user's needs, and hide data that is not required.

The dynamic tabs plugin is built to work with any entity. In GCPM 2.1, it has only been configured for amendments.

When configuring a dynamic tab record, choose an entity and form. Each section of the form can be hidden or displayed.

To configure this plugin, choose a form, and provides a check box of sections on the form. The list of check boxes are like the stages in the business process flows of hidetab, except that you can select multiple at the same time to reveal more tabs.
