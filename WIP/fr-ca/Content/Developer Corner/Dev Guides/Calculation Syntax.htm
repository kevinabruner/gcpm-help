Description

LEN

Obtient la taille d'une chaîne

ESTNULLE

Si l'expression est nulle, utiliser la valeur par défaut.

IIF

Obtient l'une des deux valeurs selon le résultat de l'expression logique

TRIM

Enlève tous les blancs avant ou après comme \r, \n, \t, ' '.

TRIM(‘ Bonjour ‘)

SUBSTRING

Obtient une sous-chaîne d'une longueur spécifique Commence à 1.
