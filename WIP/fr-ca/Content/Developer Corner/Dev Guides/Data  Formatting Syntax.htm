Guide de syntaxe pour le formatage de données

Chaînes

Il n'y a pas vraiment de formatage à l'intérieur d'une chaîne, au-delà son alignement. L'alignement fonctionne pour les arguments imprimés dans un appel String.Format.

Échantillons Génère

Échantillons

Génère
String.Format(“->{1,10}<-“, “Bonjour”); -> Bonjour<-

String.Format(“->{1,-10}<-“, “Bonjour”);
