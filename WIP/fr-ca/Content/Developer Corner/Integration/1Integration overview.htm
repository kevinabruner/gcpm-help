Fichier de conversion des codes postaux (FCCP)

Le FCCP relie un code postal à des données géographiques, comme une division de recensement, une subdivision de recensement, des coordonnées de latitude et de longitude, des circonscriptions électorales fédérales. La tarification est disponible sur demande de Postes Canada.

Les données comprennent :

  • Un code postal à six caractères
  • Un identifiant de l'aire de diffusion composé du code de la province ou du territoire, du code de division du recensement et du code de l'aire de diffusion
  • Bloc de diffusion : une région géographique de base (dans la mesure du possible)
  • Les coordonnées représentant la latitude et la longitude de la géographique du recensement auxquelles le code postal est lié
  • Le nom, le code et le type de subdivision de recensement (SDR)
  • Les codes géographiques d'autres régions géographiques normalisées de niveau supérieur dans lesquels le bloc de diffusion ou l'aire de diffusion sont situés
  • Code de circonscription électorale fédérale
