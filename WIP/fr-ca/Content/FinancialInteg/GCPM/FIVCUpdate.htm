Mettre à jour les enregistrements FournisseurClient

Les enregistrements FournisseurClient sont stocké dans le système financier de votre organisation et du GPSC. Lorsque ces enregistrement sont changés dans un système, les changements doivent être mis à jour dans l'autre système. Les flux ont été créé pour mettre à jour les données. Par exemple :

  • Si les enregistrement FournisseurClient sont gérés dans le système financier, utilisez le flux Update Account from VendorCustomer (Mettre à jour les comptes à partir de FournisseurClient) afin de mettre à jour les enregistrements du GPSC avec les changements.
  • Si les enregistrement FournisseurClient sont gérés dans le GPSC , utilisez le flux Update VendorCustomer from Account (Mettre à jour les FournisseurClient à partir des comptes) afin de mettre à jour les enregistrements du système financier avec les changements.

Remarque : ces flux suppose qu'un enregistrement d'intégration financière du FournisseurClient existe déjà. En général, le moment pour générer une intégration financière des FournisseurClient pour un nouveau Client SC est près du moment de l'approbation du projet. Le flux qui gère la génération d'intégration financière de FournisseurClient à partir d'un projet est Generate VendorCustomer from Project (Générer un FournisseurClient à partir d'un projet). Si vous souhaitez générer une intégration financière du FournisseurClient ailleurs, créez un nouveau flux modélisé sur le flux Generate VendorCustomer from Project.

Synchroniser les enregistrements FournisseurClient

  1. Ouvrez un compte et choisissez Exécuter un flux
  2. Choisissez le flux approprié à partir de la liste des flux disponibles.

  3. Cliquez sur Ajouter.
  4. Enregistrez et ferme l'enregistrement.
