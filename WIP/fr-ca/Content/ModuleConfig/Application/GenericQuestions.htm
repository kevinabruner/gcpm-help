Questions pour les demandes génériques

Les questions génériques qui s'affichent sur les demandes du portail doivent configurées à deux endroits :

  • Dans l'entité de demande. Voir ci-dessous.
  • dans le formulaire du portail Alphinat. Consulter Configurer des formulaires pour le portail.

Afin de créer des questions de demande génériques, ouvrez l'entité de Demande dans l'interface de CRM.

  1. Naviguez vers Paramètres > Solutions et choisissez la solution dans laquelle l'entité de demande est conservée.
    Par exemple, GCPM_Base.
  2. Naviguez vers Demande > Champs et créez un nouveau champ.
  3. Configurez le champ afin qu'il réponde aux besoins de la question. Notez que les options disponibles changeront, selon le type de données choisies.
     
    • Nom d'affichage : le texte affiché pour la question.
    • Exigence du champ : il indique si le champ est optionnel, recommandé pour les entreprises ou exigé pour les entreprises.
    • Nom : ce champ se rempli automatiquement.
    • Type de données : ce type de données devrait correspondre au type sélectionné dans le formulaire Alphinat. Si vous utilisez des ensembles d'option, les options devraient être identiques sur la demande et sur le formulaire Alphinat.
  4. Dans l'affichage des solutions pour la demande, naviguez vers les Formulaires. Sélectionnez le formulaire pour la demande.
  5. Tous les nouveaux champs que vous avez créé apparaissent dans l'Explorateur de champ.
  6. Créez un onglet dans l'entité de Demande. L'affichage des renseignements de la colonne peut être changé en sélectionnant le type de Colonne qui se trouve dans l'onglet du menu Insérer.
    Notez que tous les champs généraux du GPSC son saisis dans l'onglet Information générale.
  7. Afin de changer le texte d'entête de l'onglet affiché, faites un double-clic sur le nouvel onglet et modifiez les propriétés. Changez le champ  Étiquette pour le texte qui sera affiché sur le formulaire de cet onglet.
  8. Afin de nommer les sections de colonne, faites un double-clic sur la section à l'intérieur de l'onglet afin d'ouvrir les propriétés du menu.

    • Étiquette : entrez un nom d'affichage approprié.
    • Afficher l'étiquette de cette section sur le formulaire : cochez cette case.
  9. Glissez et déposez les champs dans l'onglet et la section désirés du formulaire.
  10. Faites un double-clic sur les champs pour modifier les propriétés, comme en lecture seule, verrouillage du champs et visible.
  11. Enregistrer le formulaire. Pour avoir un aperçu du formulaire avant de le publier, sélectionnez Aperçu > Bureau du client >  Créer formulaire.
  12. Lorsque le formulaire est terminé, cliquez sur Publier.
