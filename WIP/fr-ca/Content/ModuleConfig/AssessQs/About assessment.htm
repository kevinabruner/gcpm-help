À propos des questions d'évaluation

Les question d'évaluation s'assurent que la diligence raisonnable d'un agent et que l'évaluation d'un projet sont saisies et peuvent être intégrées dans les communications au client, dans des formulaires d'approbation de projet, etc.

Les modèles d'évaluation contiennent des questions pertinentes pour le programme. Les questions peuvent être réutilisées dans différents programmes et sont automatiquement copiées à des projets avec l'outils de clone.

Les questions peuvent être configurées afin qu'elles s'affichent aux emplacements appropriés sur le formulaire du projet (par exemple, des questions sur le coût s'affiche dans une grille sous la grille des coûts, etc.). Il y a 5 grilles de question pour l'évaluation par défaut: le financement du projet, l'analyse financière, les coûts du projet, les aides précédentes et les considérations pour le remboursement.

Type de données Commentaires

Question d'évaluation

Questions configurées dans le programme.

Non modifiable par les utilisateurs.

Valeur de l'évaluation

Soit un champ de texte ou une liste filtrée de réponses selon la question d'évaluation. Les utilisateurs ne peuvent que choisir une seule réponse. Peut être obligatoire ou non selon la configuration. Les utilisateurs peuvent faire des commentaires additionnels.

Commentaires

Peut être configuré pour être obligatoire selon la valeur de l'évaluation.
