Configurer un modèle d'évaluation de priorité

Les modèles de priorité s'assure que les mêmes critères de priorité sont copiés à tous les projets dans un programme (ou une demande, si le modèle est configuré pour les demandes).

Pour configurer les modèles de priorité:

  1. Dans la recherche avancée, cherchez pour Modèle de priorité | A et créez un nouvel enregistrement.
  2. Remplissez les champs en utilisant l’aide à l'écran et les notes ci-dessous.

    • Nom: fournissez un nom au modèle (anglais et français).
    • Formule de déploiement: choisissez le calcul qui note le niveau de priorité global lors de l’utilisation de ce modèle.
    • Priorités: choisissez les priorités pour ce modèle.
      Les priorités sont des éléments « parents » du modèle de priorité, qui agissent comme catégorie (par exemple, priorité = Gestion et critère = questions liées à la gestion).
    • Critères: choisissez les critères pour la priorité sélectionnée ci-dessus. Les critères sont des éléments « enfants » du modèle de priorité.
  3. Continuez à ajouter des critères jusqu’au moment où vous avez terminé le modèle.
  4. Enregistrez et ferme le modèle.
