Configurer le classement prioritaire automatisé pour les demandes

Remarque : cet article suppose que vous êtes familier avec la configuration de Modèles d'évaluation de priorité. Consultez cet ensemble d'article pour de plus amples détails : À propos de l'Évaluation prioritaire.

Le classement prioritaire automatisé permet aux réponses des clients dans une demande d'être automatiquement noté dans une évaluation de priorité. Chaque choix de réponse (stockée dans l'entité de Choix de réponse) pour la question de demande se fait attribué un score. Le score total pour une demande est calculée automatiquement pour une évaluation de priorité automatisée. Seulement les questions qui ont des choix de réponse configurés peuvent être utilisé pour un classement prioritaire automatisé.

La configuration est réalisée en deux étapes :

  1. Éléments de modèle de priorité : pour chaque élément de modèle de priorité que vous voulez remplir automatiquement, sélectionnez la question du formulaire de demande qui remplira automatiquement cet élément de priorité.
    1. Dans la grille Priorités, examinez les éléments du modèle de priorité.
    2. Pour chaque priorité qui devrait se remplir automatique à partir d'une demande, Modifier l'élément de modèle de priorité.
      Consulter Créer les éléments prioritaires d'un modèle pour des instructions plus détaillées.
  2. Valeur de la cotation :
    1. Vérifier que le champ Question de la demande est la question appropriée de la demande. Cela permet de s'assurer que le champ sera rempli automatiquement avec l'entrée de la demande du client. Vérifiez qu'une valeur de pointage est sélectionnée pour chaque Option.

      Consulter Options de demande pour des instructions plus détaillées.
    2. Le choix du pointage devrait correspondre au choix de question de la demande et à l'élément de modèle de priorité

  3. Créer une évaluation prioritaire Consulter Déterminer la priorité du projet.

    L'évaluation génère plusieurs éléments d'évaluation de priorité, s'il y a des questions de demande qui ont une réponse qui correspond à cet élément de réponse. Dans ce cas, choisir automatiquement cette valeur. Le pointage automatisé calculera le score d'évaluation de priorité. Si toutes les questions sont configurées pour se remplir automatiquement par la demande du client, vous avez une solution de classement de priorité automatisé.

Pour voir les classements de toutes les demandes dans une possibilité de financement :

  1. Naviguez à la possibilité de financement associée avec ces demandes.
  2. Dans la barre de navigation supérieure, allez à Demandes.
  3. Si l'affichage approprié n'a pas été créé, créé un aperçu dans l'entité de Demande qui contient le champ « Score calculé » du champ de recherche « Dernière évaluation ». Cela devrait ressembler à ce qui suit (recherche avancée utilisée à des fins de démonstration) :
