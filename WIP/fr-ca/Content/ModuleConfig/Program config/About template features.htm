À propose des fonctions qui utilisent des modèles

De nombreuses fonctions du GPSC (voir le tableau ci-dessous) utilisent des modèles pour s'assurer que les champs de données sont appliqués de manière uniforme à tous les projets d'un programme.

Il est possible d'utiliser deux modèles pour les clients SC – suivi et évaluation du risque.
