Mettre à jour les entités de recherche

Pour mettre à jour une entité de recherche

  1. Dans Recherche avancée, cherchez l'entité que vous souhaitez mettre à jour.
    Étiquettes est utilisé dans cet exemple.
  2. Cliquer ! Résultats. La liste des registres affiche les valeurs de cette entité de recherche.
  3. Sélectionnez un registre et le modifier, le désactiver ou le supprimer s'il y a lieu.
    Pratique exemplaire : Désactivez les registres indésirables. Ne supprimez pas les registres de recherche s'ils concernent des comptes. Les registres désactivés ne sont plus disponibles aux nouveaux registres, mais conservez les valeurs dans les registres existants.
  4. Enregistrez et fermez l'enregistrement.

Note : Les grands ensembles de données (comme les villes ou les codes SCIAN) peuvent être exporter en bloc par l'administrateur du système GPSC.

Attention!

  • La suppression des valeurs de recherche entraîne la suppression de données, si la valeur a été attribuée à un registre.
  • Par exemple, si l'étiquette « Municipalités » est supprimé, tous les comptes qui avaient l'étiquette « Municipalités » n'auront plus cette étiquette.
