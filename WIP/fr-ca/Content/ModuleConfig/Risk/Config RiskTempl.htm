Configurer un modèle de risque

Les modèles de risque s'assure que les mêmes catégories et éléments de risque sont copiés à toutes les évaluations des risques du projet dans un programme (ou un évaluation des risques du client, si le modèle est configuré pour les risques du client).

Pour configurer un modèle de risque:

  1. In the program which you want to configure risk, navigate to Project Assessment Templates section.
    If there is no pre-configured template, or you wish to create a new one, click + to add a new record.
    Sinon, naviguez à Administration des affaires > Modèles > Modèle de risque | A. Créez un nouveau registre.
  2. Remplissez les champs en utilisant l'aide à l'écran et les notes ci-dessous:
    • Type de modèle: choisissez le type de modèle.
      • Client - le modèle sera disponible pour une évaluation des risques du client.
      • Projet - le modèle sera disponible pour l'évaluation des risques du projet.
    • Statut du modèle: les options sont (En voie de construction / Compléter). Changer le statut des champs de verrouillage complétés sur le modèle.
      • Les modèles complétés peuvent être associé avec un programme ou pour les évaluations risque du Client SC.
      • En voie de construction = Tous les champs peuvent être modifiés par un configurateur. Le modèle n'est pas disponible pour la création d'un projet ou pour les évaluations risque du client .
    • Type de cotation: choisissez la façon dont la cote de risque est résumé selon la cote de risque. Ces résumés peuvent être textuel (élevé-moyen-faible) ou numérique (1-2-3-4-5). Les options communes sont disponibles, d'autres peuvent être créées. Voir Au sujet de la notation
    • Cumuls de la cotation: choisissez la formule approprié pour calculer une cote pour les catégorie dans ce modèle.
      La formule choisie offre une valeur numérique qui est un calcul sommaire des cotes de la catégorie de risque.
  3. Ouvrez la section Catégorie de risque et cliquez sur + afin d'ajouter une nouvelle catégorie de modèle de risque.
    Utilisez l'option de la grille ou du formulaire - ces captures d'écran utilise l'option du formulaire.
  4. Remplissez les champs pour la catégorie du modèle de risque en utilisant l'aide à l'écran et les notes ci-dessous:

    • Nom: donner un nom (anglais et français). Les agents verront ces noms.
    • Modèle de risque: prérempli. Cela est le modèle associé avec cette catégorie de risque
    • Catégorie de risque: choisissez une catégorie de risque à utiliser dans ce modèle.
      Ce champ affiche toutes les Catégories de risque | A déjà défini dans votre organisation. Si vous choisissez une catégorie existante, les champs de cumul et de pondération du risque afficheront les valeurs de la catégorie une fois que l'enregistrement est enregistré. Par ailleurs, vous pouvez créer une nouvelle catégorie de risque.
    • Cumul de la cote: cette option détermine la façon dont la cote de risque globale sera calculé.
      Consultez Au sujet de la notation pour de plus amples détails.
    • Pondération: (facultatif) offre une pondération de 1 ou une autre valeur si vous choisissez de pondérer la catégorie différemment.
    • Type d'élément de risque: choisissez la grille pour afficher cette catégorie de risque.
  5. Choisissez les éléments du modèle de risque.
    • Nom: donnez un nom (anglais et français). Les agents verront ces noms.

    • Modèle de risque: prérempli. Cela est le modèle associé avec cette catégorie de risque

    • Catégorie de risque: choisissez la catégorie de risque à laquelle cet élément est associé.
    • Élément de risque : Choisissez l'élément de risque de la liste déroulante associée à ce modèle d'élément de risque.
    • Pondération: (facultatif) offre une pondération de 1 ou une autre valeur si vous choisissez de pondérer la catégorie différemment.
    • Type d'élément de risque: choisissez la grille pour afficher l'élément de risque.
  6. Enregistrez et fermez l'enregistrement.
