Métadonnées du cadre d'action

Le premier tableau ci-dessous identifie les entités dans le cadre d'action. Les tables restantes affichent les métadonnées SESC pour chaque entité. Ces métadonnées ont été générées à partir de l'environnement Tools 9 le 18 juin 2019.

Nom d'affichage de l'entité Description Nom logique
CFG Actions Fonctionnalité d'action - Instance d'utilisation d'une définition d'action (p.ex.: s'assurer que la date estimée de fin dans le profile du projet n'est pas vide) egcs_action
CFG définition d'action Cadre d'action - Définition générique (ou réemployable) de la fonctionnalité qui peut-être utilisé pour une action ou une validation (p.ex.: comparer des champs) egcs_actiondefinition

Journal d'action Fonctionnalité d'action - egcs_actionlog

CFG définition des paramètres d'action Cadre d'action - Liste des paramètres liés à la définition de l'action (entrante et sortante) egcs_actionparameterdefinition
CFG Jeu d'action Cadre d'action - Groupe d'action pouvant être appelé à partir de transitions ou de JavaScript egcs_actionset
CFG condition d'action Fonctionnalité d'action - liste des valeurs qui sont utilisés comme des conditions. egcs_condition
