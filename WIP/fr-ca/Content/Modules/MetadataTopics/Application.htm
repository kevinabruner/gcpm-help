Métadonnées du module de demande

Le premier tableau ci-dessous identifie les entités dans la module de demande. Les tables restantes affichent les métadonnées SESC pour chaque entité.

Ces métadonnées ont été générées à partir de l'environnement Workspace 9 le 18 juin 2019.

Application module entity list

Nom d'affichage de l'entité Description Nom logique
Décision de demande Application feature - Delegated authorities decision on the application egcs_application_decision
Modèle de demande | A Application feature - An application form. egcs_ay_applicationform
Catégorie de la demande Application feature - A section on the application form. Allows for the grouping of similar questions together. egcs_ay_applicationformcategory
Options de demande Application feature - A valid response to a question. Defines the responses available for a question. egcs_ay_applicationformoption
Question relative à la demande | A Application feature - A question on an application form. egcs_ay_applicationformquestion
Réponses à la demande Application feature - A response to a question on the application form. egcs_fc_applicationquestion
Modèle de demande | PG Application feature - A program-specific application. egcs_tp_applicationform
