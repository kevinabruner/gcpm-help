Métadonnées du module des questions d'évaluation

Le premier tableau ci-dessous identifie les entités dans la module d'évaluation. Les tables restantes affichent les métadonnées SESC pour chaque entité.

Ces métadonnées ont été générées à partir de l'environnement Workspace 9 le 18 juin 2019.

Liste d’entités pour la module de Questions d'évaluation

Nom d'affichage de l'entité Description Nom logique
Question d'évaluation | A Une liste des questions d'évaluation utilisées par votre organisation egcs_ay_assessmentquestion
Modèle d'évaluation | A Fonctionnalité d'évaluation - Un regroupement de questions d'évaluation pour votre organisation egcs_ay_assessmentquestionstemplate
Valeur de l'évaluation Une liste des valeurs d'évaluation utilisées par votre organisation egcs_ay_assessmentvalue
Question d'évaluation | PJ Une liste des questions d'évaluation utilisées dans votre organisation egcs_fctpay_assessmentquestion
Modèle d'évaluation | PG Modèle (s) de questions d'évaluation pour un programme Les questions dans le modèle choisi sont copier dans les projets associés à ce programme. egcs_tp_assessmentquestionstemplate
Élément du modèle d'évaluation Questions d'évaluation qui font partie d'un modèle d'évaluation et qui sont copiées dans les évaluations de projet. egcs_tpay_assessmentquestion
