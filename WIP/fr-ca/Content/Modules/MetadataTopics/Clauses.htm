Métadonnées de la module clause

Le premier tableau ci-dessous identifie les entités dans la module de clause. Les tables restantes affichent les métadonnées SESC pour chaque entité. Ces métadonnées ont été générées à partir de l'environnement Workspace 9 le 18 juin 2019.

Liste d'entités

Entité Description Nom logique
Catégorie de clause Une liste des clauses d'accord de contribution utilisées dans votre organisation egcs_ay_clausecategory
Clause | A Une liste des clauses d'accord de contribution utilisées dans votre organisation egcs_ay_clauses
Modèle de clause | A Fonction de clause - Un regroupement des clauses de l'entente de contribution pour votre organisation. egcs_ay_clausestemplate
Élément du modèle de la clause Fonction de la clause - Clauses pour les ententes de contribution qui font partie d'un modèle de clause. egcs_ay_related_ay_clause
Clause | PJ Clauses d'accord de contribution disponibles pour un programme egcs_fc_profileay_clause
Modèle de clause | PG Modèles de clauses d'accord de contribution disponibles pour un programme Les clauses du modèle sélectionné sont copiées dans les projets du programme associé. egcs_tp_clausestemplate
