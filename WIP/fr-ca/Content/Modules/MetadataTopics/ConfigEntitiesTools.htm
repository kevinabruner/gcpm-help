Métadonnées des entités de configuration du GPSC

Le premier tableau ci-dessous identifie les entités de configuration GPSC qui utilisent une ou deux entités. Les tables restantes affichent les métadonnées SESC pour chaque entité.

Ces métadonnées ont été générées à partir de l'environnement Tools 9 le 18 juin 2019.

Liste d'entités

Entité Description Nom logique
CFG Bilingue Entité de configuration pour le plugiciel CFG Bilingue egcs_bilingual
CFG Calcul Entité de configuration pour le plugiciel des champs calculés egcs_calculatedfield
CFG Plugin Override   egcs_cfg_pluginoverride
CFG Clone Entité de configuration pour le plugiciel clone egcs_cloneconfiguration
CFG clone liés Entité de configuration pour cloner un groupe tel qu'un modèle ou une liste de contrôle. egcs_clonerelatedconfiguration
CFG de détection en double Outil de configuration de détection en double - entité de configuration pour créer des règles de détection en double au délà des règles en double fournit dans le produit CRM. egcs_duplicatedetection
CFG onglet dynamique Outil de configuration - Entité de configuration pour le plugiciel onglet dynamique egcs_dynamictabconfiguration
CFG Verouillage Une configuration Entité pour le plugiciel Verouillage Les enregistrements contiennent une configuration principale pour le verrouillage des champs et des grilles d’enregistrements CFG Champ verrouillage associés. egcs_lockconfiguration
CFG Champ verouillage Une entité de configuration pour le plugiciel verrouillage. Holds records with a value of a target entity's lookup or option set and the list of fields (locked field list) that lock if this value is selected on the record being checked. egcs_lockfieldconfiguration
