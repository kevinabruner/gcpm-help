Métadonnées du cadre de décision-transition

Le premier tableau ci-dessous identifie les entités dans le cadre de décision-transition. Les tables restantes affichent les métadonnées SESC pour chaque entité.

Ces métadonnées ont été générées à partir de l'environnement Tools 9 le 18 juin 2019.

Decision-transition framework entity list

Entité Description Nom logique
Action d'approbation   egcs_approvalaction
Activité d'approbation Approval feature - egcs_approvalactivity
CFG Authority Authority framework - these records create a relation between an authority set and an authority type egcs_authority
CFG Authority Pathway Authority framework - Authority pathways represent one step in an authority set egcs_authoritypathway
CFG Authority Type Authority framework - these records create a relation between a type of approval and roles egcs_authoritytype
CFG Authority Set Authority framework - CFG Authority Set records bring together multiple approvers in appropriate order to form an approval chain egcs_ay_authorityset
Type de décision Decision feature - A list of the types of decisions that can be made egcs_ay_decisiontype
Type de document de décision Decision feature - Documents to be generated with a decision. Used to specify more than one document per decision egcs_ay_decisiontypedoc
Statut   egcs_ay_status
Décision du projet Project feature - A list of decisions that can be made for projects egcs_fc_decision
Historique du projet Project feature - egcs_disb_transitionhistory
Transition Path Transition feature - Configures the transitions for a project workflow egcs_fc_transitionstatusid
Modèle de transition Transition feature - Includes all possible transition paths for a business process flow egcs_fc_transitiontemplateid
