Métadonnées du gestion de documents

Le premier tableau ci-dessous identifie les entités dans la module de gestion de document. Les tables restantes affichent les métadonnées SESC pour chaque entité. Ces métadonnées ont été générées à partir de l'environnement Tools 9 le 18 juin 2019.

Document manager tool entity list

Entité Description Nom logique
CFG d'entité du document A configuration entity that contains document management configuration for a specific entity and document repository. egcs_cfg_documententity
CFG du lieu du document A configuration entity that represents a folder or a predefined location in the external document repository. egcs_cfg_documentlocation
CFG Type de documents A configuration entity that identifies a document as a specific type of document. egcs_cfg_documenttype
CFG du champ du document A configuration entity that represents a single metadata field in the document repository that maps to a field in CRM. egcs_cfg_metadatafield
GPSC Document Un document dans le dépôt de document egcs_document
GPSC Lieu du document Document manager feature - An entity that identifies a folder for the storing of a document. egcs_documentlocation
GPSC version du document This entity has not been implemented egcs_documentversion
