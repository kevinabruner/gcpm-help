Métadonnées de la module de performance

Le premier tableau ci-dessous identifie les entités dans la module de performance. Les tables restantes affichent les métadonnées SESC pour chaque entité. Ces métadonnées ont été générées à partir de l'environnement Workspace 9 le 18 juin 2019.

Performance module entity list

Entité Description Nom logique
Indicateur du projet | A Fonctionnalité liée à la performance - Une liste d'indicateurs de projet utilisés par votre organisation. egcs_fc_indicator
Résultat du projet Fonctionnalité liée à la performance - les résultats de performance liée a un projet. egcs_fc_projectresult
Résultat du programme Fonctionnalité liée à la performance - Une liste des résultats associés aux programmes prioritaires. egcs_pp_outcome
Programme de priorité | A Fonctionnalité liée à la performance - Une liste de programmes prioritaires pour votre organisation. egcs_pp_program
Indicateur du projet | PJ Fonctionnalité liée à la performance - Une liste d'indicateurs de projet associés à un projet. egcs_ppfc_indicator
Programme prioritaire | PG Fonctionnalité liée à la performance - Une liste de programmes prioritaires pour un programme. egcs_pptp_program
Historique du résultat du projet Fonctionnalité liée à la performance - Historique lié aux résultats d'un indicateur pour un projet. egcs_projectresulthistory
