À propos du portail

Le portail est un site Web SharePoint qui sert de point d’accès frontal Internet au GPSC. Il s'agit d'un outil de communication entre une agence de développement régional (ADR) et les internautes, offrant une solution de rechange au téléphone, au courriel et aux documents papier. Le portail permet a votre organisation de recueillir les renseignements que les clients présentent dans le portail, de traiter les présentations dans le GPSC et d'en publier les résultats dans le portail pour un examen par les clients. Les clients peuvent surveiller l'état de leurs présentations et de leurs projets avec votre organisation.

Communications

Le portail prend en charge trois types de communications :

  • La messagerie sécurisée entre le client et votre organisation. Solution de rechange plus sûre que le courriel, où le contenu du message reste privé dans le portail et dans le GPSC.
  • Présentation de formulaires : Demandes, réclamations, avances et rapports sur le rendement. De nouveaux formulaires peuvent être ajoutés au portail. Les formulaires qui sont présentés dans le portail déclenchent automatiquement des processus opérationnels dans le GPSC (p. ex. demandes). Le moteur de formulaire pour le portail est SmartGuide d'Alphinat.
  • Documents partagés. Des documents arbitraires peuvent être téléchargés dans le portail afin d'être partagés avec l'ADR. Ces documents sont automatiquement copiés dans le GPSC.

Zones

Le portail prend en charge trois zones. Toutes les communications énumérées ci sont disponibles dans chaque zone, même si ce n'est pas chaque formulaire qui sera disponible dans chaque zone.

  • Zone personnelle : Quiconque sur Internet peut s'inscrire pour obtenir un compte dans le portail. Chaque utilisateur se voit attribuer sa propre zone personnelle. Le principal objectif de la zone personnelle est la communication sécurisée avec l'ADR. Cela permet a un utilisateur de commencer à traiter avec son ADR. Les formulaires disponibles comprennent les demandes.
  • Zone Client. Un client est une personne morale  (et non un particulier) qui traite avec votre organisation. Les utilisateurs peuvent être invités à une Zone Client dans des rôles divers  (lecture, modification, administration). Un utilisateur peut passer en revue des renseignements sommaires sur son client, ainsi que l'état d'une demande pour le client. La  Zone Client comporte aussi des liens vers les projets qui ont été publiés dans le portail (voir Zone Projet). Les utilisateurs administrateurs peuvent inviter d'autres utilisateurs du portail à se joindre à leur client. Les utilisateurs invités peuvent être en mesure de lire et de modifier des documents et des messages sécurisés ainsi que de présenter des formulaires pour un client et ses projets. Les formulaires disponibles comprennent les demandes.
  • Zone Projet : Une Zone Projet représente une entente de contribution unique entre un client et votre organisation. Elle affiche des renseignements financiers sur le projet (comme les renseignements sur le budget ou le calendrier de remboursement pour le projet). Une Zone Client peut comporter plusieurs zones projet. Les formulaires disponibles comprennent les réclamations, les avances et les rapports sur le rendement.

Toute communication présentée d'une zone devient associée à cette zone dans le GPSC.

Par exemple  : un message sécurisé envoyé d'une Zone Client s'affichera sous ce client dans le GPSC. Une réclamation présentée d'un projet deviendra une réclamation pour ce projet en particulier.
