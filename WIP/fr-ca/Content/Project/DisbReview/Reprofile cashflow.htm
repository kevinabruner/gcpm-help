Reporter un flux de trésorerie

Pendant la phase d'administration de projet d'un projet, le client présente les demandes de paiements des coûts (réclamations et paiements anticipés).

Lorsque vous faites un examen de débours, vous pouvez changer les éléments de paiements ou ajuster (reporter) le flux de trésorerie. Les ajustements peuvent être requis s'il y a des retards dans le projet ou si le client ne dépensent pas les fonds aussi rapidement que prévu à l'origine.

Pour reporter un flux de trésorerie pour un projet :

  1. Dans un examen des débours, naviguez vers la grille des Flux de trésorerie et déterminer quels ajustements doivent être apportés.
  2. Ouvrez le projet. Naviguez à Administration du projet > Modifications et lancez une nouvelle Modification administrative.
  3. Vous pouvez modifier le flux de trésorerie entre de multiples années ou ajouter un nouvel exercice financier pour le flux de trésorerie.
    Remarque : le flux de trésorerie doit s'équilibrer avec le financement approuvé par le ministère. Si le flux de trésorerie non attribué n'est pas 0 $, la modification ne peut pas être recommandé jusqu'à ce que des modifications soient apportées pour obtenir un montant nul.
  4. Lorsque la modification administrative est approuvée, la grille de résumé des flux de trésorerie sera automatiquement mise à jour.
