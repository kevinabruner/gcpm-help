Suivi d'un projet

Lorsqu'un projet en est à l'étape du suivi, les agents peuvent surveiller les activités de suivi au moyen de l'outil de suivi de projet.

Pour ajouter une activité de suivi à un projet :

  1. Dans un projet, naviguer au chevron Administration de projet, et ouvrez l'onglet Suivi de projet.
  2. Cliquez sur +NOUVEAU pour ajouter un nouveau suivi du projet.
  3. Remplir les champs au moyen de la fonction d’aide à l'écran et des notes ci-dessous.
    • Statut du suivi: indique le statut (en cours / compléter)
    • Type de suivi : choisissez la fréquence (annuelle / semestrielle / trimestrielle / mensuelle / ponctuelle).
    • Modifier la fréquence : si vous souhaitez choisir la fréquence des activités de suivi futures, choisissez une option pertinente. Cette fréquence sera en vigueur pour les activités de suivi futures.
    • Remarques: offre une remarque, au besoin.
    • Prochaine date de revue: entrez la date de la prochaine activité de suivi.
    • Date d'achèvement : ce champ se remplit automatiquement quand le statut du suivi est changé pour « Compléter ». Cette date peut être écrasée manuellement au besoin.
  4. (facultatif) Ajoutez un élément de l'examen afin de saisir les notes occasionnelles pour supplémenter l'activité de suivi. Il est à noter qu'il s'agit d'une activité du CRM qui peut être affichée dans les tableaux de bord. Remplissez les champs au moyen de la fonction d'aide à l'écran et les notes ci-dessous :
    • Sujet : identifiez le sujet de cet élément d'examen.
    • Date de suivi : indiquez s'il y a une date pour faire le suivi de l'élément.
    • Sur la bonne voie : indiquez si l'élément est sur la bonne voie (oui / non).
    • Description : fournissez une description de l'élément.
    • Commentaires : fournissez des commentaires sur les progrès de cet élément, ou d'autre contenu pertinent.
      Remarque : votre organisation peut élargir l'utilisation de ces éléments d'examen, afin de les afficher à titre de rappel sur le tableau de bord de l'agent ou une liste de tâches.
  5. Répondez aux questions de la liste de contrôle pendez que vous effectuez l'activité de suivi. Les questions sont particulières au programme.

  6. Lorsque le suivi est complété, changez le statut à Complété.

  7. Si l'examen financier est requis dans le cadre de l'activité de suivi, cliquez sur + pour ajouter un nouvel examen.
  8. Enregistrer et fermer l'enregistrement.
