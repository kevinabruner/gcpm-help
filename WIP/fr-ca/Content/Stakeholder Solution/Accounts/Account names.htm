Noms de comptes

Les registres de comptes comporte des champs pour les noms d'exploitation et les noms légaux. Le suivi des modifications au nom légal se fait à la fonction d'historique du nom légal.

Noms d'exploitation

Le nom d'exploitation d'une organisation est employé pour les activités quotidiennes et à des fins publicitaires. Dans la GPSC, un nom d'exploitation est requis.

  • Lorsque vous saisissez le nom d'exploitation (AN), il est automatiquement copié dans le champ Nom d'exploitation (FR) (et vice versa).
  • S'il y a des noms d'exploitation en français et en anglais, saisissez-les dans leur champ respectif.
  • S'il y a seulement un nom d'exploitation, saisissez-le dans la langue appropriée. Le nom est automatiquement copié dans le champ de l'autre langue.
  • S'il n'y a pas de nom d'exploitation, le nom légal s'affiche dans les champs du nom légal et du nom d'exploitation.
  • Le nom par défaut d'un compte (utilisé dans les recherches et les rapports) est le Nom d'exploitation (AN).
  • Un administrateur du système GPSC peut modifier le champs du nom pour afficher le nom d'exploitation (FR) ou le nom légal.
    Consultez : Configurer les noms des comptes.
Noms légaux

Le nom légal d'une organisation est indiqué dans le certificat de constitution, de fusion, de prorogation ou de modification. Une personne morale peut inscrire au registre un nom légal à la fois en anglais et en français.

  • S'il y a un seul nom légal, saisissez-le dans le champ de la langue appropriée. Le nom est automatiquement copié dans le champ de l'autre langue.
  • S'il y a un nom d'exploitation différent en français et en anglais, entrer les noms dans les champs appropriés.
  • Le nom légal utilisé dans les recherches et les rapports est le nom légal (AN).
  • Le suivi des modifications au nom légal se fait à la fonction d'historique du nom légal lorsque l'identité de l'entreprises est validée.
  • Voir de plus amples renseignements ici : Modifier le nom légal
Valider l'identité d'une entreprise

Le non légal peut être saisi par un client, à l'aide du portail, ou par un membre du personnel. Le nom légal devrait être validé avant qu'un accord de contribution puisse être créé pour le client. Servez-vous de Identité de l'entreprise validée pour indiquer que cette recherche a été faite.

Il est à noter que Corporations Canada offre une orientation pour faire des recherches pour les nom légaux qui sont dans les deux langues*.
