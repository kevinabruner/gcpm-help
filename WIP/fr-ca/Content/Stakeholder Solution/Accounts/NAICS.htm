Codes SCIAN

Pour attribuer des codes SCIAN à un compte :

  1. Dans l'enregistrement du compte, ouvrez la section Profil du compte > SCIAN.

  2. Cliquez sur l'icône de recherche et choisissez un code SCIAN approprié.
  3. Afin de sélectionner des codes SCIAN supplémentaires, il faut utiliser la grille Autre et cliquer sur l'icône +.
    Il est également possible de cliquer sur l'icône SCIAN dans le plan du site du compte.
  4. Sauvegardez et fermez l'enregistrement.

Remarques :

  • Vous pouvez saisir les chiffres initiaux du code ou les mots utilisés dans le nom du code afin de trouver rapidement le code approprié.
  • Les codes SCIAN peuvent servir aux rapports et aux tableaux de bord.
  • Le Système de classification des industries de l'Amérique du Nord (SCIAN) est un système de classification utilisé par le Canada, le Mexique et les États-Unis pour organiser les données économiques en fonction des secteurs d'activité.
  • Le système GPSC est peuplé avec la version la plus récente de Statistique Canada, Canada 2017 version 2.0. La prochaine version est prévue en 2022.

  • Le SCIAN utilise des enregistrements hiérarchisés. Les enregistrements du SCIAN sont en relation les uns avec les autres en fonction du nombre de chiffres. Par exemple, 5412 est un groupe relevant de 541 – Services professionnels, scientifiques et techniques.
