Télécharger le document

Générer un document qui se situe à l'intérieur d'une configuration du générateur de document. Une fois que le document est généré, un score est créé avec le document en tant que pièce jointe dans le projet. La fonction se base sur trois paramètres et retourne le document généré :

Configuration : la configuration du générateur de document.

Modèle : le modèle que nous utiliserons pour générer le document.

Enregistrement : il enregistre les données utilisées pour généré le paramètre documentReturn :

Document : le document généré.

Envoyer à : le champ qui contient le destinataire du courriel. La recherche est généralement liée à un utilisateur qui a un courriel.

Envoyer par : le champ qui contient l'auteur du courriel. La recherche est généralement liée à un utilisateur qui a un courriel.
