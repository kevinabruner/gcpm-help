Configurer les enregistrements bilingues

Les champs de la GPSC sont configurés afin de s'afficher en anglais ou en français, selon la langue de choix de l'utilisateur. Un utilisateur choisit la langue de choix aux réglages de l'utilisateur*.

Pour configurer les champs afin d'afficher la langue de choix de l'utilisateur :

  1. Identifier le champ qui doit s'afficher dans la langue de choix de l'utilisateur. Noter le nom de l'entité et le nom de champ logique.
  2. Naviguer a Configuration >  Bilingue > CFG Bilingue. Créer un enregistrement CFG Bilingue pour le champ.
  3. Remplir les champs au moyen de la fonction d'aide à l'écran et des informations ci-dessous.
    • Le Nom n'a pas de format, mais il doit être unique.
      Conseil : utiliser une norme, comme le nom d'une entité à traduire suivi par le mot traduction. Dans l'exemple ci-dessus, Nom = Type d'entreprise traduction
    • Entité cible : le nom logique de l'entité où se trouve le champ.
      Dans notre exemple, entité cible = egcs_sh_businessstructure
      Remarque : ce champ est sensible à la casse.
    • Champs à traduire : une liste de champs à traduire délimitée par des virgules (utiliser un nom logique).
  4. Enregistrez et fermez l'enregistrement.

CFG Bilingue travaille de concert avec le module d'extension CFG Calcul afin d'afficher la langue de l'utilisateur pour les champs bilingues, au besoin. Voir À propos de l'outil de configuration bilingue:

Exemple

Par exemple, les enregistrements de type d'entreprise comprennent les noms anglais et français pour les types d'entreprise:

  • Nom AN=University
  • Nom FR=Université

Le mot « university » s'affiche si l'utilisateur a l'option de langue anglaise et « Université » s'affiche si l'utilisateur sélectionne l'option de langue française.

Remarques :

  • Le plugiciel affiche les champs traduits qui ont un symbole de tuyau | délimitant les termes anglais et français. Si un champ n'a pas de symbole de tuyau, il ne peut pas afficher le texte anglais / français.
  • CFG Bilingue se déclenche chaque fois que l'entité et que le champ sont chargés.
