À propos du générateur de documents

Le générateur de documents peut générer tout document dont votre organisation peut avoir besoin pour gérer les clients et les produits. Les utilisateurs professionnels peuvent modifier tout modèle de document afin d'afficher les données du CRM. Le générateur de documents est un outil qui prend des instantanés de données et qui les convertit en format pouvant être lu par des humains (p. ex., un document, un rapport ou une page Web). Le générateur de documents offre un formatage et une conception de contenu flexibles, et il peut facilement personnaliser des documents pour faire état de l'aspect et de la convivialité de votre organisation.

Il est possible de configurer les modèles de document afin d'afficher des renseignements propres à un projet. Un contexte supplémentaire, comme les clauses déterminées par les résultats d'évaluation et les données de projet, sont configurées au moyen d'un code XML qui est lié aux champs du GPSC.

Voici des exemples de clauses conditionnelles : Il y a trois clauses liées aux règlements environnementaux, mais seulement une clause peut être choisie pour tout projet donné, en fonction des caractéristiques du projet. Lorsqu'un agent identifie une option convenable pour un projet, la clause appropriée est insérée dans EC pour ce projet.

Un configurateur définit les champs de données a inclure dans un instantané, et l'instantané remplit un modèle de document MS Word avec des données sous format XML.

Le GPSC comprend un modèle EC configuré.

Fonctions :

  • Créer des instantanés : (en formats XML ou JSON)
    • Une vue des données à un moment précis
    • La base pour générer des documents (ou des rapports) 
  • Générer des documents (en formats Word, HTML, PDF)
Technologies requises :
  • Mappage XML Microsoft Word (disponible dans MS Word 2013 et versions plus récentes)

Outils utiles (il est possible d'en utiliser d'autres) :

  • Open XML SDK : bibliothèque de source ouverte de Microsoft qui vous permet de créer et de modifier de façon programmatique des documents Office sans avoir à installer Microsoft Office.
  • Open XML PowerTools : bibliothèque de source ouverte de MS qui ajoute des fonctions supplémentaires à Open XML SDK, comme la création et l'assemblage de documents et la conversion du format Word au format HTML
  • XPath 2.0 : utilisé par OpenXML PowerTools afin d'insérer du contenu dynamique dans MS Word.
