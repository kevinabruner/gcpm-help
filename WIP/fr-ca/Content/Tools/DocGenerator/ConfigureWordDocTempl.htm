Configurer un modèle de document Word pour un instantané

Un instantané de données (créé avec CFG image de l'instantanée) peut être inséré dans un document  Word. Le document Word doit être configuré au moyen du XML afin d'afficher les données GPSC.

Pour configurer un document Word XML : 
Note : nécessite Microsoft Windows 2013 (ou version plus récente).

  1. Affichez l'onglet Développeur dans Microsoft Word.

    Si vous ne voyez pas l'onglet Développeur, ouvrez Fichier > Options, sélectionnez Personnaliser le ruban sous Onglets principaux, cochez la case Développeur, puis cliquez sur OK. L'option Développeur apparaît dans  le ruban Word.
  2. Configurer un document pour l'entente de contribution, incluant toutes les sources de données nécessaires.
  3. Une fois qu'un instantané a été configuré, dans l'enregistrement CFG Snapshot, cliquez sur MODÈLE WORD, puis choisissez l'option pour télécharger un modèle vierge.

  4. Enregistrez le fichier. Le modèle de document vierge contient les données XML que vous pouvez insérer et référencer dans votre modèle de document.
    Vous pouvez maintenant formater le modèle avec le texte et les champs appropriés.
  5. Pour voir les champs disponibles dans le modèle, choisissez Developpeur – mappage XML. Le panneau mappage XML s'ouvre à la droite. Seules les données de l’entité précisées dans les enregistrements sources de données de l’instantané seront disponibles dans le modèle de document.
  6. Ajoutez le texte pour ce modèle et le format afin de répondre à vos besoins.
  7. Pour insérer une valeur de champ, déplacez votre curseur à l’endroit où vous voulez l’insérer. Trouvez le champ dans le panneau de mappage XML, faites un clic droite sur le champ, et sélectionnez l'option pour insérer en texte brut.
    Consultez les conseils ci-dessous pour plus d’idées sur le mappage de champs.

  8. Sauvegardez le modèle localement.

  9. Créez un nouvel enregistrement CFG Modèles de document, et téléchargez le modèle configuré.
    Le modèle apparaît dans l’onglet Modèles de l'enregistrement CFG Modèles de document.

  10. Associez l'enregistrement CFG Modèles de document à un programme (ou à l’entité appropriée).

Conseils sur l’insertion de champs

  • Le champ peut être formaté comme vous le voulez – en gras, en italique, souligné, etc.
  • Pour les champs qui ressemblent à des paragraphes (p. ex., un champ ayant beaucoup de texte), utilisez du texte enrichi plutôt que du texte brut.
  • Certains champs standard peuvent être insérés dans le modèle. Par exemple, pour créer l'horodatage sur le document créé à partir du modèle :
    • @currentdate inserts the date the document is created.
    • @currenttime insère l'heure de création du document.
  • Les entités et les champs sont affichés avec des noms logiques, et non avec des noms affichage. Familiarisez-vous avec les noms logiques pour vous assurer de choisir les bons champs.
  • L’insertion d’un « dossier » comme données ne produira rien dans le document. p. ex., référencer ou insérer le nom du compte associé à un projet : 
  • Les données de champ n’affichent pas de flèche de navigation du côté gauche, parce qu’il s’agit d’une seule donnée, et non d’un « dossier » qui contient des données.
