Principales étapes pour utiliser le générateur de documents

Pour générer un document qui insère des données du GPSC dans un modèle de document :

  1. Décidez si vous pouvez utiliser le modèle de document CRM ou si vous devez utiliser le générateur de documents.
    (insérez les considérations)
    • Si vous choisissez les modèles de document CRM :Utilisation de modèles de document
    • Sinon, poursuivez ces étapes
  2. Configurer les sources de données pour un instantané
  3. Configurer un document
  4. Configurer un modèle de document Word pour un instantané pour afficher les données du  GPSC.

  5. Créez une configuration de générateur de documents.

  6. Téléchargez les modèles de document selon la configuration.
