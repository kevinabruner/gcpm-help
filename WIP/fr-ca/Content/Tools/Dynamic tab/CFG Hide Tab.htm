Masquer les onglets de formulaire dans les flux de processus d'entreprise

Vous pouvez configurer les onglets de formulaire afin qu'ils soient masqués à différentes étapes d'un flux de processus d'entreprise. C'est une façon utile de personnaliser l'expérience de l'utilisateur afin qu'elle montre seulement les sections pertinentes du formulaire à une étape donnée d'un flux de processus d'entreprise.

Masquer les onglets de formulaire

Configurer les onglets de formulaires afin qu'ils soient masqué pendant un flux de processus d'entreprise :

  1. Naviguez à Configuration > Masquer onglet et cliquez sur CFG Masquer onglet.
  2. Créer un nouvel enregistrement CFG Masquer onglet et remplissez les champs en utilisant l'aide à l'écran.
    • Flux de processus d'entreprise : choisissez l'un des flux de processus d'entreprise disponibles.
      Remarque : les étapes dans le flux de processus d'entreprise s'affichent dans la rangée du haut du tableau.
    • Formulaire : choisissez un formulaire disponible dans la liste déroulante.
      Remarque : tous les formulaires pour les entités comprises dans le flux de processus d'entreprise sélectionné sont affiché dans la liste déroulante. Chaque onglet du formulaire sélectionné s'affiche dans la première colonne du tableau ci-dessous.
  3. Afin de masquer un onglet durant une étape, enlever le crochet dans la case appropriée. Si la case est cochée, l'onglet sera affiché.
    Conseil : choisissez la première case dans la rangée pour afficher un onglet de formulaires à toutes les étapes du flux de processus d'entreprise.
  4. Cliquez sur Enregistrer configuration. Une fenêtre flottante affiche un message de confirmation.

  5. Pour mettre à l'essai la configuration, entrez un enregistrement et changez les étapes en cliquant sur elles, Les onglets devraient se masquer ou s'afficher automatiquement selon la configuration.

Afficher un formulaire à afficher dans un flux de processus d'entreprise

Configurer un formulaire à afficher dans le mode du concepteur.

  1. Ouvrez le formulaire sélectionné dans le mode du concepteur.

  2. Ajoutez la ressource Web egcs_hidetab_initialization.js web resource to the form and call the init function on the form’s OnLoad du formulaire (si elle n'est pas déjà configurée de cette façon).

  3. Enregistrez et fermez le formulaire et publiez vos changements.

Le formulaire devrait maintenant s'afficher selon vos paramètres lorsque le flux de processus d'entreprise est amorcé.
