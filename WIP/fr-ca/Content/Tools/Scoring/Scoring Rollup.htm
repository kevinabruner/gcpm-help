Configurer des cumuls de notation

Les cumuls de notation offrent des options de calcul des notes en fonction des valeurs des éléments et des critères.

Ces options de cumuls courants sont incluses dans le GPSC.
  • Moyen
  • Moyen modifiable
  • Somme modifiable (simple - valeurs seulement)
  • Somme modifiable (simple avec facteur de pondération)
  • Maximum
  • Minimum
  • Addition

Lorsqu'un cumul des notations est utilisé avec un gabarit d'évaluation, la notation de l'évaluation est calculée à l'aide du cumul sélectionné pour les notes d'éléments de risque. Dans certains cas, un administrateur peut annuler manuellement la notation calculée, dans la mesure où il en donne une justification. (p.ex., des évaluations de risques).

Pour configurer un nouveau cumul de notations :

  1. Dans recherche avancée, chercher Cumul de la cotationet créer un nouvel enregistrement de cumul.

  2. Nom : saisir un nom (anglais et français) pour le cumul.

  3. Cumul : choisir une option de cumul [Somme, Moyenne, Valeur minimale, Valeur maximale]
  4. Formule : Si souhaité, créer un formule pour calculer le cumul. Voir les notes ci-dessous.
  5. Enregistrez et fermez l'enregistrement.

Remarques :

  • Pour créer une nouvelle formule :

    Le champ Formule utilise les variables que voici :

    • (valeur) - regroupe les éléments de risque saisis dans l'évaluation
    • (pondération) - configure le facteur de pondération de la question (ID) utilisé pour compter le nombre des éléments dans cette catégorie

    Créer d'autres variables dans le champ Expressions de formules.

    Expression de formules : Saisir de nouvelles variables à utiliser dans la formule dans le champ ci-dessus.

  • Options de cumul : Dans Microsoft Dynamics, les attributs du cumul (comme la somme, la moyenne, la valeur minimale, la valeur maximale) des calculs configurés préalablement et dont vous pouvez vous servir dans la formule. Lire plus au sujet de ces calculs ici* :

  • Formule : sum(total)

  • Variable : total = facteur de pondération * valeur

  • Les expressions de formules permettent de créer des variables qui peuvent être utilisées dans la formule. Comme le code de plugiciel ne permet pas des expressions à l'intérieur d'une fonction d'agrégat (somme, moyenne, etc.), vous pouvez créer ces variables, puis les insérer dans la formule.

  • Par exemple : Formule : x2 + 2y = z

    • Variables :

      • X = 2

      • Y = 3

    • Formule : sum(total)

    • Variable : total = facteur de pondération * valeur
