Configure action definitions

Action definitions are (short blurb)

To configure a new action definition: 

  1. In Advanced Find, search for CFG Action Definition and create a new record.
  2. Name (EN, FR): Type a name for the action
    (Tips or best practice?)
  3. Process: Choose a process.
    The process may be a workflow or action. In this example, the download document process is displayed:
  4. Function name: What are rules or tips for the name provided here?
  5. Type: Indicate if this configuration is an action or validation.Explain the difference-why does it matter?
  6. Description: Type a description to indicate the purpose of this action definition.
  7. Messages: Type the message that will appear to the end user if the action definition has an error.

Parameter definitions

  1. Choose (or only create?) action definitions to populate the Parameter Definitions grid.
  2. Save and close the CFG Action Definition record.
