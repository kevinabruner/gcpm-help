Using the rollup plugin

This is an example using the rollup plugin (CFG Rollup) to calculate project costs.

There are four rollup fields in project costs: 

  • Total cost items (count)
  • Total eligible costs
  • Total costs
  • Total contribution

Parent: project costs

Child: project cost items

The Assistance rate field is a CRM field that calculates the percentage of the Total ACOA contribution / Total eligible costs

To define a rollup field: 

  1. Create a new CFG Rollup record.
    This example uses egcs_project costs. It shows the rollup configuration for the Total ACOA Contribution rollup field in project costs. The other three rollup fields contain similar configurations, but point to different rollup and aggregate fields.

    The value of this field is a FetchXML query string. You can use CRM’s advanced find to build a FetchXML query string. For more complex queries, you can use a third party tool called FetchXML Builder*.

    This field leverages NET Framework’s library to perform the data formatting and concatenation.

    The syntax for formatting and concatenating fields is the same as the String Format(String format, params object[] args) method in the .NET Framework, where this field is the String format parameter and Formula Parameters is the object[] args parameter.
