Logic Operators
Specifier Description Example Result
=

Equal 1 = 1

1 = 1 True
<> Not Equal 2 <> 1 True
< Less Than 2 < 1 False
<= Less Than or Equal To 2 <= 2 True
> Greater Than 2 > 1 True
>= Greater Than or Equal To 1 >= 2 False
AND Whether both conditions are true 1 < 2 AND 2 < 10 True
OR Whether at least one of the conditions is true. 1 < -1 AND -1 < 10 True
NOT Makes a condition opposite. NOT 2 < 1 True
IN Whether a value is in a set of values 2 IN (1, 3, 5, 7, 9) False
LIKE LIKE Whether a string is like a pattern. ‘John Doe’ LIKE ‘%John%’ True
IS NULL Whether a value is null or not. null IS NULL True
BETWEEN Whether a value falls within a range 15 BETWEEN 10 AND 20 True
* or % Wildcards. Cannot be used in the middle of a string. Wildcard characters must be escaped by square brackets, same with square brackets. ‘%[%][[]Hello[]][%]%’  
( and ) Parentheses. Group comparisons together. (1 = 1 AND 1 = 2) OR 1 = 1 True
Functions
Specifier Description Example Result

Gets length of string.

ISNULL If expression is null, use default value. ISNULL(null, 0) 0
IIF Gets one of two values depending on the result of a logical expression. IIF(true, 1, 0) 1
TRIM Removes all leading and trailing blank characters like \r, \n, \t, ' '. TRIM(‘ Hello ‘) Hello
SUBSTRING

Gets a substring of a specified length. Starts from 1.

SUBSTRING(‘Hello World’, 7, 5) World
