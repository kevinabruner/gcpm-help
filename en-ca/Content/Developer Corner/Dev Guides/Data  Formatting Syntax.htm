Data formatting syntax guide

Strings

There really isn’t any formatting within a string, beyond its alignment. Alignment works for any argument being printed in a String.Format call.

Sample Generates

Sample Generates
String.Format(“->{1,10}<-“, “Hello”); -> Hello<-

String.Format(“->{1,-10}<-“, “Hello”);

->Hello <-
