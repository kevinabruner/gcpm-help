Creating a new EditAble CRM Grid

 

 

EditAble CRM Grids on the CRM Site Map

 

To create an EditAble CRM Grid, or manage an existing one, go to Settings > EditAble CRM Grids in your CRM organization.

 

 

Create a New EditAble CRM Grid

To create a new EditAble CRM Grid Configuration, simply click the new button to launch the EditAble CRM Grid Configuration Wizard which will guide you through the basic setup of an EditAble CRM Grid.

Step 1

Enter a name for the EditAble CRM Grid - This is how you will refer to the grid when adding it to CRM.  The name should be unique among the other EditAble CRM Grid configurations in your organization.

Select the entity to show in the grid - This is the CRM entity that you want to display in the grid.  For example, if you are creating an EditAble CRM Grid to show Open Opportunities, you would select "Opportunity".

Select a mode for the grid - Select if you want the rows in the grid to be editable or read-only.

 

Step 2

In step 2 you can select the entity's attributes that you want to appear on the EditAble CRM Grid as columns. Attributes on this page can be sorted by Display Name, Schema Name, and Data Type. Only attributes from the entity selected in Step 1 are available on this page.  You will be able to add attributes from related entities on the main configuration form.

 

Step 3

Next, you'll see a summary of the columns you selected in Step 2.  You can configure the label and width for each column, or change the order that the columns appear by dragging the column's Name field to a different position.

You will be able to make additional changes to the columns in the main configuration form.

 

Step 4

 

Select how the records should be filtered.

Show records ... 

·Related to a parent record - Selecting this option will show all records related to the entity you select in "Select the parent record entity".  This option will typically be used when creating an EditAble CRM Grid that is going to be placed on a CRM form (like a sub-grid).  It is similar to how a CRM Associated View works.

·Based on specific criteria - This option will not add predefined filters.  You can use the Criteria builder to specify the filters you want in the EditAble CRM Grid.

 

Step 5

Step 5 summarizes the options you have selected.  When you click the Finish button, the EditAble CRM Grid configuration will be created.

 

After you've set up the basics of your EditAble CRM Grid, you can make changes to the Row Functionality & Display Options, as well as, how the grid is Grouped & Sorted. There are several other advanced options and the ability to set filters and custom events.
