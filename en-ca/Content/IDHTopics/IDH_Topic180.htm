Previewing an EditAble CRM Grid

 

 

If you click “Preview”, from the ribbon, you can see how the EditAble CRM Grid will look on a form.

 

 

 

This can be very useful when creating new EditAble CRM Grid configurations or making changes to existing ones.  However, it is very important to note that even the though EditAble CRM Grid is in a "preview" mode, it is using live data from your CRM organization.

 

 

 

The "Select Parent Record" field is available for previewing EditAble CRM Grids that are configured as "Related to a parent parent".  You can change the value of the parent record to see how it will appear for different parent records.

 

Tip: The Preview window does not use the published configuration, which means all you need to do is save your configuration changes and the Preview window will show the latest updates.  This also means that you can make modifications to your EditAble CRM Grid configuration without affecting the published version that you may have already added to your CRM organization and users may be using.  Using the Preview functionality, you can get the configuration where you want it and then publish to update the version user's see.  Note that if you have Auto-Publish enabled for your configuration, it will be published automatically anyway.
