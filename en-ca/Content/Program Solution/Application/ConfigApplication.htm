Configure an application

To configure an application in GCPM: 

  1. In Advanced Find, search for ApplicationAn application is submitted by a client in response to funding opportunities. Applications must be separately configured for the portal and GCPM. Portal configuration includes establshing how online application are transferred to GCPM. and create a new record.
  2. Fill in the fields using onscreen help and the notes below: 
    • Application / project ID: A unique ID for the application, separate from the project ID. Is it auto-generated?
    • Application template: choose the template this application is associated with.
    • Application status: indicates the status of the application. (how can the status be changed - action set? other trigger?)
    • Submission type: does applicant choose this option? I thought the options here would be portal / manual
    • Date submitted: auto-populated?
    • Submitted by: would be auto-populated if submitted on the portal? If manually input, can I switch to GC client or account lookup, or would it be a "portal user" account created for manual submission? Or would I be able to choose the portal user?
    • Account: populated manually?
    • GC Client: populated manually?
    • Funding opportunity: auto-populated based on template?
    • Project: once this is populated, are any other fields populated or overridden?
    • Program: auto-populated based on template?
    • Account legal name: should this be grouped with other client info? e.g. Account, GC client lookup...
    • Project name: What impact does filling in this field have - is it copied on to project? 
  3. General Information (should this read: client information?)
    • are the fields here (e.g. business number, sector, city, province...) to hold data submitted via portal? How do they link to GCPM fields?
  4. Applicant Responses to Questions: a completed application lists the questions (from the template) as well as the respondent's answers to the questions.

  5. Assessments: any priority assessments that have been completed for this application are listed in the Priority Assessments grid.
