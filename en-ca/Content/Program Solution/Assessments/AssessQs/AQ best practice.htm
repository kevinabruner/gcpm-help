Best practices: assessment questions

It is a best practice to map out your assessment process before finalizing assessment questions, to take advantage of the automation and efficiencies that assessment questions can provide.

Display questions in appropriate locations

Assessment questions can be displayed in a single section on an assessment form, or they can be sorted into grids (according to section title) below relevant sections. For example, questions on cost items can be displayed below the cost item table; questions on financing displayed below the financing table, etc.

  • Configure this using the Section title field.
  • Grids to display the questions must be configured in each section of the form where the assessment questions will be displayed. Crowe grids require that text responses are configured in a different grid from lookup value responses.

Display data in GCPM based on assessment results (Verify with Anthony)

  • Using the Question ID field, questions can be used to automatically show / hide form sections based on the response to the question.
  • For example, if the question "Project costs are over $5M" is "No" then the section on the project assessment form that contains questions for projects over $5M can be hidden.
Backlog / future option: 

Control which questions appear on projects. Additional questions can be displayed on projects that meet specific criteria.

For example, the question, "Is the organization for-profit or non-profit?" can be included to determine if the recipient is a for-profit entity.

  • If the answer = for-profit, questions that pertain to for-profit entities can be displayed in a separate Crowe grid from questions limited to for for-profit entities.

  • If the answer = non-profit, the for-profit questions can be hidden
  • Sections can be configured to show or hide based on the response to the "For-profit / non-profit" question.
  • Alternatively, grids can be configured to appear in different forms (e.g. one form for for-profit, one form for non-profit). (VERIFY)
Reference assessment results in documents

Assessment questions can be used to ensure an officer's assessment is reflected in communications with the client, the contribution agreement and into the monitoring phase of the project, if applicable.

  • The ID field is alphanumeric (can be made up of letters and numbers) but should not contain a space.
  • Reference the ID field in XML
    For example, the question "Does the project require a change of control?" is assigned question ID = "COC". When designing the contribution agreement, a reference to "COC" in the XML template will insert the appropriate clause for the response to this question.
  • When configuring the contribution agreement, the clause associated with change of control can be inserted with the reference "COC"
When to create multiple assessment templates

It is possible to design one assessment template to address different assessment factors, rather than creating multiple assessment templates. Using CRM info, questions can be configured to display only when a project meets certain criteria (e.g. repayable or non-repayable project). However, if there are more than one or two conditional assessment factors, this can lead to complex configuration for a single template.

If there are more, it is advised to make separate templates. For example: 

  • Project costs are over / under $5M

  • For-profit and non-profit entities
