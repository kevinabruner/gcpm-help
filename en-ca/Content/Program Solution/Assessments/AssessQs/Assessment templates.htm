Configure an assessment question template

Assessment templates ensures a consistent application of the same assessment questions to every project in a given program. See this backgrounder: About features that use templates.

To configure assessment templates:

  1. Populate the lookup entities that are used in the assessment template: 
    Assessment question
    Assessment values
  2. In Advanced Find, search for Assessment Question Template | A and create a new record.
    Alternatively, navigate to the Business Administration tile and choose the Assessment Q icon under the Templates heading.

  3. Name: Type the name of the template.
  4. Assessment questions: Click the + icon to add questions to this template.

    Choose a question and configure how it will work in this template by completing these fields: 
    • Template: Displays the template you are configuring.
    • Assessment question: Choose from one of the questions available
    • Question type: Indicate if the question must be answered by an officer (required) or if it is optional (discretionary)
    • Mandatory: Indicate if the question is mandatory.
    • Answer type: Indicate if the answer provided is text, or a measurement value.
    • Question ID: Provide an alphanumeric (text and/or number) ID to use the response to this question in automated features. See Best practices: assessment questions for details.
    • Section title: Choose the section where this question should appear.
  5. Save and close the template.
