Configurer un enregistrement de CFG Cumul

Le CFG Cumul fonctionne avec le plugin de cumul afin de faire des calculs spécifiques (aussi appelés cumul) au besoin.

Pour configurer un enregistrement de cumul:

  1. Dans recherche avancée, chercher CFG Cumul et créer un nouvel enregistrement de cumul.

  2. Utiliser l'aide à l'écran en tant que guide afin de remplir ces champs. Notez que quelques champs ont une aide additionnelle, ci-dessous.
  3. Enregistrer et fermer l'enregistrement

Pour le champ type de regroupement, ces valeurs sont liés à chaque étiquette.

Les champs de l’entité de cumul énumérés ci-dessous sont utilisés à l'interne (ils ne sont pas afficher sur la forme) afin de pré-traiter les actions dispendieuse dans le plugin :

Champ de configuration

Description

Interrogation

(Caché) La chaîne d'interrogation FetchXML pour le calcul de cumul.

Entités liées

(Caché) Les entités parents qui sont mentionnés dans les paramètres de la formule.

Métadonnées

(Caché) Les métadonnées de calcul sont entreposé ici afin d'optimiser le rendement.

NOTE : * Le champ de cumul et le champ de regroupement doivent avoir le même type de données!
