Dynamics 365 Help

Configurer un enregistrement de CFG Séquence

Dynamics 365 a une capacité de numérotation automatique dès sa sortie de l'emballage, qui génère un id unique à incrémentation automatique pour un enregistrement. Par contre, cette caractéristique ne peut être utilisé que pour certaines entités.

Le générateur de séquence GPSC (egcs_sequencegenerator) duplique la fonctionnalité de numérotation automatique et il peut être utilisé pour toutes entités.

Voir aussi : À propos du plugin CFG Séquence.

Formule

La séquence dynamique d'incrémentation automatique pour générer un nouvel enregistrement. Ce champ tire parti de la bibliothèque de .NET Framework pour accomplis le formatage et la concaténation des données.

La syntaxe pour les champs de formatage et de concaténation est la même que la méthode du « String Format » (Format de la chaîne) dans .NET Framework, où le champ est le paramètre du String Format et les Paramètres de la Formule sont le paramètre object[] args.

Voir la documentation sur la méthode du format pour une syntaxe valide de formatage des données. Un aide-mémoire est fourni.

La formule générale est : Faire référence aux attributs dans les Paramètres de la Formule, suivre la convention fictive String.Format : {0} pour référer au premier attribut, {1} pour le deuxième et ainsi de suite. Le nombre maximum doit être moins ou égal au nombre des attributs énumérés dans les Paramètres de la Formule. Les nombres fictifs peuvent être utiliser plus d'une fois dans la formule.
