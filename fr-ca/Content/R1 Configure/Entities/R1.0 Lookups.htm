    Notes sur le remplissage des valeurs de recherche « Villes » :

    • L'entité de recherche des villes s'assure de l'intégrité des données en limitant les utilisateurs à choisir des noms de lieux à partir d'une liste, à la place de les entrer manuellement. Vous pouvez remplir cette entité avec la liste de villes de votre choix.
    • Une source de données recommandée est le fichier de données de Noms de lieux de Postes Canada. Veuillez noter que ce fichier contient de multiples noms pour le même lieu, afin d'accommoder les différents types d'orthographe.
      Par exemple, St. John, St-John, St John.
    • Si vous choisissez d'utiliser le fichiers des Noms de lieux de Postes Canada, assurez-vous qu'il n'y ait pas de lieux en double avant de téléverser les données à l'entité Villes.
    • Le fichier des noms de lieux contient aussi la province, la division de recensement et la subdivision de recensement pour chaque nom de lieu. Vous souhaiterez peut-être remplir les champs avec ces données. Les améliorations pour comprendre ce type de fonctionnalité peuvent être compris dans les futures publications.
    • Vous pouvez télécharger vos propres fichiers de noms de lieux ici : http://geosuite.statcan.gc.ca/geosuite/en/index#self ou ici.
    • Pour en savoir plus sur les produits de Postes Canada : Intégrer la GPSC aux outils d'adresse de Postes Canada
