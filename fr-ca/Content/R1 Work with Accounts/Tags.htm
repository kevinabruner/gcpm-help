Ajouter une étiquette a un compte

Pour ajouter une étiquette à un compte :

  1. Dans un enregistrement de compte, ouvrir la section Étiquette.
    La section Étiquette fait partie de l'onglet Profile du compte. Une grille affiche les étiquettes actuelles.

  2. Cliquer + pour ouvrir un nouveau formulaire Étiquette-Compte, et remplir les champs. Utiliser l'aide qui s'affiche comme guide.
    Remarque : le Nom est généré automatiquement lorsque vous choisissez une Étiquette.
  3. Sélectionner une étiquette. Vous pouvez, facultativement, fournir une date de début et de fin, ainsi que des commentaires.
  4. Enregistrer et fermer l'étiquette.

Remarques :

  • Les étiquettes doivent être ajoutées ou supprimées manuellement des comptes.
  • Les étiquettes sont une façon utile et rapide de créer une collection de comptes avec des attributs communs. Une fois que l'étiquette a été appliquée aux comptes, vous pouvez utiliser l'étiquette pour produire des rapports sur ces comptes.

  • Le configurateur de la  GPSC peut localiser les étiquettes par défaut fournies dans la GPSC afin de respecter vos besoins.

  • Les étiquettes de compte sont stockées dans l'entité Étiquettes (nom du schéma : egcs_SH_AccountGroupType).

Exemple :

  • Ajouter l'étiquette Universités à tous les comptes d'universités.
  • Créer un rapport en fonction de l'étiquette Universités.
  • Vous pouvez créer une liste marketing dynamique sur les comptes avec une étiquette, ou une combinaison d'étiquettes.
